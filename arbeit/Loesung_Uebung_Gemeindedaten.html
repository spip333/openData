Einleitung

Ziel dieser Übung ist es, die im Modul „Graphische Datenexploration und Datenvisualisierung“ vermittelten Techniken anzuwenden. Grundlage dafür ist ein Datensatz mit dem regionalen Portrait der Schweizer Gemeinden des Jahres 2014, der über das Open-Data Portal des Bundes frei zugänglich ist (https://opendata.swiss/de/dataset/regionalportrats-2014-kennzahlen-aller-gemeinden/resource/63a4f255-28f3-4568-8e3a-75e3aa466e6b). Der auf Moodle abgelegte Datensatz ist bereits mit einzelnen Merkmalen zur räumlichen Gliederung der Schweiz ergänzt: - Kanton - Grossregionen der Schweiz - Sprachgebiete - Städtische/Ländliche Gebiete

Zusatzerläuterungen zu den Daten finden Sie in den Dokumenten: - be-b-00.04-rgs-15.xls (Raumgliederung Schweiz) - je-d-21.03.01.xls (Kennzahlen Regionalportrait)

Es handelt sich um einen Datensatz, der nicht explizit für Analysezwecke aufbereitet ist, vielmehr ist das Regionalportrait eine Sammlung unterschiedlichster Kennzahlen. Eine erste Übertragung in ein einheitliches Datenformat ist im File gemeindedaten.csv vorgenommen. Ansonsten sind die Daten aber frisch aus dem Netz. Was können wir daraus über die Gemeinden in der Schweiz in Erfahrung bringen?

5. In welchem Kanton gibt es am meisten Gemeinden? In welchem am wenigsten?

# Anzahl Gemeinden je Kanton als Daten
gemeinden<-
  gemeindedaten %>%
  group_by(kantone)%>%
  summarise(count = n_distinct(bfsid))

ggplot(gemeinden,aes(x=reorder(kantone,count),y=count))+
  geom_bar(stat="identity",color="white",fill="#687d8e",color="grey50")+
  coord_flip()+
  geom_text(aes(label=count),hjust=1.25,colour="white",size=2.75)+
  labs(title="Anzahl CH-Gemeinden nach Kanton",subtitle="2014", y="Häufigkeiten",x="Kantone CH",

10. Erstellen Sie ein politisches Profil nach Sprachregionen mit der Hilfe der Variablen zu den Wähleranteilen.

# Daten aufbereiten
# Parteien-Variablen
library(tidyr) # für gather()
profil_sprachregionen <- gemeindedaten %>%
  select(sprachregionen,starts_with("polit")) %>%
  group_by(sprachregionen) %>%
  summarise_all(funs(mean(.,na.rm =TRUE))) %>%
  gather(variable,value,-sprachregionen) %>% # Braucht es hier um das spezifisch benötigte Format zu erhalten
  arrange(variable)

# Data-Polishing
profil_sprachregionen[is.na(profil_sprachregionen)]<-0 # NA führen zu Problem
profil_sprachregionen$variable<-gsub("polit_","",profil_sprachregionen$variable) # Parteien-Variable ohne _polit

# Nun kommen die Radar-Charts
# Mit einem Facet_wrap
ggplot(profil_sprachregionen) +
  aes(x=variable, y=value, group=sprachregionen, color=sprachregionen) + 
  geom_polygon(fill=NA) + 
  geom_point(shape=21)+
  coord_polar() + facet_wrap(~ sprachregionen) + 
  theme(axis.text.x = element_text(size = 10))+
  theme(legend.position = "none")+
  labs(title="Politische Profile CH-Gemeinden \n nach Sprachregion",
       subtitle="Nationalratswahlen 2015",
       y="Wähleranteile der Parteien in %",
       x="",
         caption="Quelle:Kennzahlen Regionalporträts-BFS")
